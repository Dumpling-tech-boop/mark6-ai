# 🧠 Mark6 AI Pro 1.0

一个基于 AI 的六合彩号码预测和数据分析工具。

## 📋 功能特性

- 📊 **仪表盘** - 实时数据状态概览
- 📂 **数据管理** - 支持手动输入、CSV 导入/导出
- 📈 **数据分析** - 热冷号统计、频率分析
- 🤖 **AI 选号** - 智能号码推荐
- ⚙️ **设置** - 自定义参数配置

## 🚀 快速开始

### 安装

无需安装，直接在浏览器中打开 `index.html` 即可。

### 基本使用

1. **查看仪表盘**
   - 点击 "📊 仪表盘" 查看当前数据状态

2. **导入数据**
   - 方式一：粘贴 JSON 格式数据到文本框
   - 方式二：导入 CSV 文件（格式见下方）

3. **分析数据**
   - 点击 "📈 数据分析" 查看号码统计

4. **生成推荐**
   - 点击 "🤖 AI 选号" 中的"生成推荐"按钮

## 📄 CSV 文件格式

```csv
期数,号码1,号码2,号码3,号码4,号码5,号码6,特码
001,5,12,18,24,31,42,7
002,3,15,22,29,35,48,14
003,8,16,25,33,40,45,21
```

## 🔧 数据格式 (JSON)

粘贴数据时使用以下格式：

```json
{
  "issue": "001",
  "numbers": [5, 12, 18, 24, 31, 42],
  "special": 7
}
```

或数组形式（直接是 7 个号码）：

```json
[5, 12, 18, 24, 31, 42, 7]
```

## 🎯 功能模块

### 1. 数据管理 (`csv.js`)
- `importCSV(file)` - 导入 CSV 文件
- `parseCSV(text)` - 解析 CSV 文本
- `exportCSV()` - 导出数据为 CSV
- `clearData()` - 清空本地数据

### 2. 数据分析 (`Chart.js`)
- `getNumberFrequency()` - 获取号码出现频率
- `getHotNumbers(topN)` - 获取热号
- `getColdNumbers(topN)` - 获取冷号
- `analyzeDistribution()` - 分析号码分布
- `generateAIRecommendation()` - AI 推荐号码

### 3. 后端接口 (`api.js`)
- `fetchLatestDrawResults()` - 获取最新开奖数据
- `fetchHistoryData(limit)` - 获取历史数据
- `getAIRecommendationFromServer()` - 获取 AI 推荐
- `healthCheck()` - 检查 API 服务状态

## 💾 数据存储

应用使用浏览器的 `localStorage` 存储数据，容量通常为 5-10MB。

**存储位置**：`localStorage.mark6_data`
