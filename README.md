# 🧠 Mark6 AI Pro 1.0

一个基于 AI 的六合彩号码预测和数据分析工具。

## 📋 功能特性

- 📊 **仪表盘** - 实时数据状态概览
- 📂 **数据管理** - 支持手动输入、CSV 导入/导出、数据去重
- 📈 **数据分析** - 热冷号统计、频率分析、遗漏号码、连续对分析
- 🤖 **AI 选号** - 智能号码推荐
- ⚙️ **设置** - 自定义参数配置、API配置

## 🚀 快速开始

### 安装

无需安装，直接在浏览器中打开 `index.html` 即可。

### 基本使用

1. **查看仪表盘**
   - 点击 "📊 仪表盘" 查看当前数据状态

2. **导入数据**
   - 方式一：粘贴 JSON 格式数据到文本框
   - 方式二：导入 CSV 文件（格式见下方）
   - 支持自动去重和数据验证

3. **分析数据**
   - 点击 "📈 数据分析" 查看号码统计
   - 查看热号、冷号、遗漏号码等详细分析

4. **生成推荐**
   - 点击 "🤖 AI 选号" 中的"生成推荐"按钮
   - 或使用 "AI智能推荐" 获得基于数据的推荐

## 📄 CSV 文件格式

```csv
期数,号码1,号码2,号码3,号码4,号码5,号码6,特码
001,5,12,18,24,31,42,7
002,3,15,22,29,35,48,14
003,8,16,25,33,40,45,21
```

## 🔧 数据格式 (JSON)

粘贴数据时使用以下格式：

```json
{
  "issue": "001",
  "numbers": [5, 12, 18, 24, 31, 42],
  "special": 7
}
```

或数组形式（直接是 7 个号码）：

```json
[5, 12, 18, 24, 31, 42, 7]
```

## 🎯 功能模块详解

### 1. 数据管理 (`csv.js`)
- `importCSV(file)` - 导入 CSV 文件
- `parseCSV(text)` - 解析 CSV 文本
- `exportCSV()` - 导出数据为 CSV
- `clearData()` - 清空本地数据
- `getStorageInfo()` - 获取本地存储信息

**新增功能：**
- 自动去重处理
- 数据格式验证
- 存储容量管理
- JSDoc文档

### 2. 应用核心 (`app.js`)
- `showPage()` - 页面导航
- `loadData()` - 数据加载
- `saveData()` - 数据保存
- `validateData()` - 数据验证
- `deduplicateData()` - 数据去重

**新增功能：**
- 全局错误处理
- 输入数据验证
- 自动去重处理
- 更新容量限制为100条

### 3. 数据分析 (`Chart.js`)
- `getNumberFrequency()` - 获取号码出现频率
- `getHotNumbers(topN)` - 获取热号
- `getColdNumbers(topN)` - 获取冷号
- `analyzeDistribution()` - 分析号码分布
- `generateAIRecommendation()` - AI 推荐号码

**新增功能：**
- `getMissingNumbers()` - 获取遗漏号码
- `getConsecutivePairs()` - 获取连续号码对
- 完整的错误处理
- JSDoc文档

### 4. 后端接口 (`api.js`)
- `fetchLatestDrawResults()` - 获取最新开奖数据
- `fetchHistoryData(limit)` - 获取历史数据
- `getAIRecommendationFromServer()` - 获取 AI 推荐
- `healthCheck()` - 检查 API 服务状态

**新增功能：**
- URL格式验证
- 超时处理
- 请求超时设置（5秒）
- 自动初始化
- 完整的错误处理
- JSDoc文档

## 💾 数据存储

应用使用浏览器的 `localStorage` 存储数据，容量通常为 5-10MB。

**存储位置**：`localStorage.mark6_data`

**存储管理：**
- 自动检测存储空间
- 超出5MB时自动清理旧数据
- 支持数据去重
- 显示存储大小
